# Campaign Performance Analytics Dashboard

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/drive/17sYyNRvUyj_FlaQpkbim0z67v4CjuJxE?usp=sharing)

Interactive dashboard for analyzing marketing campaign performance, KPIs, and uplift metrics using Python.

## Project Overview

This project builds an interactive analytics dashboard that allows marketing teams to explore campaign performance using multiple filters such as:

- Marketing Channel
- Campaign Objective
- Customer Segment
- Minimum Uplift Threshold
- Date Range

The dashboard generates insights through visualizations and KPI summaries to support data-driven marketing decisions.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- ipywidgets
- Jupyter Notebook
